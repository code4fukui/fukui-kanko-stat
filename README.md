## Features
- **Comprehensive Data Visualization:** Analyze survey data through interactive pie charts, line graphs, and bar charts with filters for demographics, location, and date ranges.
- **Trend & Correlation Analysis:** Track daily survey response counts alongside local weather data. Correlate Instagram hashtag popularity with survey response trends.
- **Geospatial Analysis:** Visualize tourist comments on a map and explore sojourning population density on a 1km mesh grid.
- **Satisfaction Rankings:** Rank tourist areas based on visitor satisfaction scores. View a "favorable impression" ranking of Fukui based on visitors' home prefectures.
- **Comment Insights:** Browse the latest survey comments in chronological order or perform keyword searches. Filter comments by tourist area, visitor origin, and age group.
- **Linked Data Exploration:** Analyze linked payment data from the RENEW Pay system in conjunction with survey responses.
